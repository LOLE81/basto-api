# Bastó Api

Backend made for a challenge, using Node.js with Express. MongoDB (mongoose) for data base. 

<br>  

In the project directory, you can run to start the server:

### `npm run dev`

<br>
<br>

For db connections check you can run tests in mocha/chai:

### `npm test`
